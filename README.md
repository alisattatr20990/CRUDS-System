# 🛒 CRUDS System - Product Management

A fully functional Product Management System (CRUD) featuring a clean, interactive UI built with core web technologies and the power of **Tailwind CSS**.

## 🚀 Features
- **Create**: Add new products with full details (Name, Price, Taxes, Ads, Discount).
- **Read**: Display products in an organized table with automatic total calculation.
- **Delete**: Remove specific items or clear the entire database with a single click.
- **Auto-Calculation**: Real-time "Total" calculation as you type numbers.
- **Local Storage**: Persistent data storage in the browser so your data isn't lost on refresh.
- **Responsive Design**: Fully mobile-friendly interface thanks to Tailwind CSS.



## 🛠️ Tech Stack
- **HTML5**: For semantic page structure.
- **Tailwind CSS**: For modern, utility-first styling.
- **JavaScript (ES6+)**: For logic, DOM manipulation, and data management.
- **Local Storage API**: For client-side data persistence.
